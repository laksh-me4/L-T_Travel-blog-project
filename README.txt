WANDER AND WRITE - Travel Blog Template

How to use:
-----------
- Place your blog images in /images/
- Open index.html in your browser.
- All styling is in /css/blog.css (use relative units, edit colors, fonts, etc.)
- Replace demo content as needed.

Features:
---------
- Responsive (mobile first), readable, clear
- Featured post section
- Grid of recent posts (cards)
- Sidebar with bio + categories
- Footer with Instagram mini gallery + newsletter
- "Back to Top" button (footer)
- Hover glow/scale on cards
- Modular, maintainable CSS classes

Customize as you like!
